## ✨ 什麼是 Purrtea？

Purrtea 是一個可愛、實用、軟呼呼的 AI 聊天介面～

**特色：**
- 🐾 **超可愛介面**：奶茶色系 + 貓咪元素
- ☕ **珍珠奶茶風格**：氣泡背景、流暢動畫
- 🎨 **兩種主題**：奶茶模式 🧋 + 夜間模式 🌙
- 🤖 **支援多種 AI 模型**：GPT、Claude、Gemini 等
- 💾 **本地儲存**：所有對話儲存在你的瀏覽器
- 📱 **全平台支援**：桌面、平板、手機都好用

## 🔑 設定 API Key

Purrtea 支援兩種 API 來源：

### 🌐 OpenRouter（推薦）

**優點：**
- 一個 Key 支援多家模型（GPT、Claude、Gemini...）
- 彈性計費，用多少算多少
- 免費試用額度

**步驟：**
1. 前往 [openrouter.ai](https://openrouter.ai/)
2. 註冊帳號
3. 創建 API Key（格式：`sk-or-...`）
4. 在 Purrtea 設定中貼上

### 🤖 OpenAI（直連）

**步驟：**
1. 前往 [platform.openai.com](https://platform.openai.com/)
2. 創建 API Key（格式：`sk-...`）
3. 在 Purrtea 設定中貼上

## 🎨 功能介紹

### 🐾 語氣模式（Prompt）

Purrtea 內建三種語氣模式：

- **劇場**：加入動作與情緒描寫
- **工程師**：精準、實作導向
- **吐槽**：毒舌但實用

你也可以自訂新的 Prompt！

### 💬 聊天管理

- 多聊天室支援
- 自動/手動命名
- 匯出對話為 Markdown
- 刪除單則訊息或整個聊天室

### 🎨 視覺特效

- 🐱 貓咪呼嚕呼嚕動畫
- ☕ 珍珠奶茶氣泡背景
- 💕 訊息飛入動畫
- 🐾 小貓掌送出按鈕

## 💡 常見問題

### Q: API Key 會外洩嗎？

**不會！** 所有資料都儲存在你的瀏覽器，沒有傳送到任何伺服器。

### Q: 對話會遺失嗎？

對話儲存在瀏覽器的 localStorage，除非你：
- 清除瀏覽器資料
- 使用無痕模式
- 手動刪除

**建議：** 重要對話可以用「匯出」功能存成 Markdown。

### Q: 為什麼叫 Purrtea？

Purr（貓咪呼嚕聲）+ Tea（奶茶）= Purrtea ☕

像貓咪陪伴一樣溫暖舒服的 AI 對話～
