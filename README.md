# 🐈 Whisper AI - 奶茶貓貓聊天介面

> 可愛、軟呼呼、互動也真的會做事喵。

一個支援多家 AI 模型的聊天介面，採用溫暖的奶茶色系設計，讓 AI 對話變得更可愛更舒服～

![Version](https://img.shields.io/badge/version-6.4.0-orange)
![License](https://img.shields.io/badge/license-MIT-blue)

---

## ✨ 特色功能

- 🎨 **奶茶色系 UI** - 溫暖舒適的視覺設計
- 🤖 **多模型支援** - GPT-5.1, Claude Opus 4.6, Gemini 2.5 Pro 等
- 🌓 **日夜雙主題** - 奶茶模式 & 夜間模式
- 💾 **對話管理** - 多聊天室、匯出對話、重新命名
- 📱 **響應式設計** - 支援桌面、平板、手機
- 🎭 **Prompt 系統** - 預設劇場、工程師、吐槽等角色
- 🔄 **即時切換模型** - 隨時更換 AI 模型
- 📋 **Markdown 渲染** - 支援程式碼高亮、表格等

## 🚀 快速開始

### 方法 1: 直接使用（最簡單）

1. 下載 `whisper-openrouter-v640-updated.html`
2. 用瀏覽器打開
3. 點選「⚙️ 設定」填入 API Key
4. 開始聊天喵～

### 方法 2: 透過 GitHub Pages

訪問：`https://你的用戶名.github.io/whisper-ai/`

## 🔑 API Key 申請

### OpenRouter（推薦）
- 網站：https://openrouter.ai
- 一個 Key 支援多家模型（GPT、Claude、Gemini 等）
- 格式：`sk-or-...`

### OpenAI 直連
- 網站：https://platform.openai.com
- 只支援 GPT 系列模型
- 格式：`sk-...`

## 📖 使用說明

### 基本操作

1. **開新聊天**：點擊左側「➕ 開新聊天」
2. **切換模型**：設定 → 選擇模型 → 儲存
3. **選擇 Prompt**：點擊上方的 Prompt 標籤（劇場、工程師、吐槽）
4. **匯出對話**：點擊「💾 匯出」下載 Markdown 格式

### 快捷鍵

- `Enter` - 送出訊息
- `Shift + Enter` - 換行

### 字體大小調整

點擊右上角的 `A-` / `A` / `A+` 按鈕

## 🛠️ 進階功能

### 自訂 Prompt

1. 設定 → Prompt 編輯
2. 點擊「➕ 新 Prompt」
3. 輸入名稱和內容
4. 儲存即可在上方標籤看到

### 管理聊天室

- **重新命名**：點擊聊天室旁的 ✏️ 圖示
- **刪除聊天室**：點擊聊天室旁的 ✕ 圖示
- **清空當前聊天**：設定 → 清空目前聊天

---

## 💡 使用建議

### 選擇模型的技巧

- **GPT-5.1 / Claude Opus 4.6**：最強大，適合複雜任務、創作
- **GPT-4o / Claude Sonnet 4.6**：平衡效能與成本
- **Mini / Haiku**：快速簡單的任務

### Prompt 使用場景

- **劇場**：創意寫作、角色扮演
- **工程師**：程式設計、技術問題
- **吐槽**：輕鬆聊天、娛樂互動

## 🔒 隱私與安全

- ✅ 所有資料儲存在瀏覽器本地（localStorage）
